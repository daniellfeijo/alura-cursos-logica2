Os navegadores entendem que o onclick é apenas para cliques do botão esquerdo! Note que, quando você clica com o botão direito, uma janelinha com o menu de ações é aberto. A função que cuida disso é a do oncontextmenu e, para impedir o navegador de abrir a janelinha de menu, você ainda pode adicionar a chamada a evento.preventDefault(): tela.onclick = function(evento){ console.log("botao esquerdo"); ... }; tela.oncontextmenu = function (evento) { console.log("botao direito"); ... evento.preventDefault(); };
